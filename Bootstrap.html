Bootstrap

On 21 Feb 2017

Bootstrap is a framework that helps creating website quickly with premade templates. It is mobile-friendly (responsive design). Bootstrap has predefined classes, which makes it easier for page layout. It enables a lot of project with its large library of features.

Boostrap Grid

Bootstrap is good for responsive design due to its mechanics with columns. Common Boostrap structure consists of 12 columns, which can be modified via CSS. With Bootstrap, containers will adjust to either the size of viewport or media query breakpoints, depending on the type of container (fixed or fluid).

It’s possible to add more than one breakpoint into the same element (class).

You can use built-in clearfix class or some responsive utility classes to fix problem with grid/columns.

Offset: “ask” certain columns to move over by a certain number of column widths. It is useful but it can affect other break points. So if you want to use it, you need to specify the modifying values at other break points as well.

Bootstrap has something called "nesting", meaning a type of "row-within-row" layout that allows for more flexible layout.

Controlling column order: columns can lie in different positions depending on the break points. You can control the order by ‘push’ and ‘pull’ in Bootstrap. Also, remember to 'pull' an element when 'push' another to keep the layout balance.

col-xs-push-1 Push the element back by 1 column at the xs (extra-small) break point.
col-sm-pull-8 Pull the element forward by 8 columns at the sm (small) break point.
 

Basic Boostrap Classes

Bootstrap uses Helvetica Neue or Helvetica. If they are not available then it will pick Arial. If even Arial is not available, default CSS sans-serif font will be used as last resort.

Global default font size for body is 14px, line height ~1.4288, and paragraph bottom margin = 10px.

Bootstrap has built-in styles for traditional inline elements, thus overriding default styles.

Some commonly seen classes include:

text-left
Text alignment. Override CSS modification. Other alignments are right, center, justify.
text-nowrap
Text lasts on one line without line break.
text-lowercase
Force lowercase. Can be used for uppercase and capitalization too.
unstyle
Remove list styling.
dl-horizontal
Horizontal definition list.
pre-scrollable
A class for <pre> tag that set the box to 300px in height and make it scrollable.

Other CSS styles

Elements that can look like buttons: links, button tag, and input fields.

Class btn btn-default: default button. Only button elements are gonna work within nav and navbar components.

Different btn other than default: primary, success, info, warning, danger.

Button sizes: default, lg (large), sm (small), xs (extra small), block (full width of parent element).

Parameter active: indicate that button is active (pressed down).

Parameter disabled: disabled button.

Class table: create table. Table-striped: every other row will have bground color. Table-bordered: create border around the table. Table-hover: hightlight when cursor is hovered over each row. Table-condensed: table taking up less space. >>> contextual classes. <tr>: table row. <td>: table cell. table-responsive: as the name suggests, setting the element wrapped around table with this class will make the table responsive to screen size.

img-responsive: auto-resize the image depending on the view port. This is a block (center-block to center the image on smaller view port). img-rounded gives rounded edge on the image. Img-circle gives circular shape.

text-muted: text appears a little lighter than normal.

Styling form

Four types of form

Bootstrap provides simple markup and styles for four styles of common web forms.

Name Class Description
Vertical (default) .form-vertical (not required) Stacked, left-aligned labels over controls
Inline .form-inline Left-aligned label and inline-block controls for compact style
Search .form-search Extra-rounded text input for a typical search aesthetic
Horizontal .form-horizontal Float left, right-aligned labels on same line as controls
