Becoming a Web Developer

On 12 Jan 2017

I'm familiar with the basic front-end term like CSS, HTML, JavaScript and GitHub since I took CIT15 last semester. I've heard of MySQL as a database management tool; Python, PHP, Java and C# as programming languages. Aside from those, I got clueless for the rest of the video. Web developing is so huge that what I know is only the tip of an iceberg.

What is Bootstrap? A Beginner's Guide

I looked up Bootstrap because I heard of it once or twice, but I don't really know what it is. So Bootstrap is a really popular framework that is used in responsive web design. Responsive design is to make web content fits into any viewing screen. No matter what device you're on, the webpage will display fully and artistically the same. If content is water, then no matter which container (devices) you pour it into, the amount of water still stays unchanged. Learn more

Node.js - Introduction

I got curious about Node.js because I saw it in my laptop's program list. Node.js is intended for developing server-side and networking applications. It's not a JavaScript framework, but it's written in JavaScript. Since it's lightweight and efficient at handling data-intensive real-time application, it's applied in real-time web applications like browser games or real-time communication programs. Learn more

Introduction to Yeoman

"Yeoman is an opinionated workflow for creating beautiful, compelling web apps."It's a scaffolding tool to create web applications. Basically, it will generate templates that you need to create a new app. It can work with any programming language. Learn more
