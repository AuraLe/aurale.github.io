Web Developer and Web-related Jobs

On 15 Jan 2017

I searched for web design job on Craigslist in Fresno/Madera area and below are a few job postings that I found. While searching on craigslist, I noticed that there are not that many openings for web-related jobs in Fresno, but there are a lot of jobs hiring in San Luis Obispo.

Entry Level Programmer

DPS Telecom - Fresno, CA | Learn more

  • Compensation: 11-15HR + Bonuses & Benefits
  • Employment type: Full-time
  • Job requirements:
    • Familiarity with CSS, PHP, HTML, Javascript, MySQL, LAMP structure, Linuz, Apache, and have worked with servers.
    • Ability to effectively communicate across departments (you will be fielding change requests and may need to provide basic training on new features).
    • Punctual and able to prioritize in order to meet deadlines.
    • Not required, but would be beneficial to be versed, in C, C++, and Pascal.
  • Prior programming experience is NOT necessarily required for this position. We can teach you the skills. We're looking for exceptionally clever people with the drive and desire to succeed. There are opportunities for advancement from this position if you demonstrate that you're a potential leader who can succeed from day one.

Web Developer

etna interactive - San Luis Obispo, CA | Learn more

  • Compensation: DoE
  • Employment type: Full-time
  • Description: Etna Interactive is seeking a Web Developer with at least 2 years of related experience. This role works closely with the creative teams to build beautifully designed, highly functional websites. We are looking for someone who is passionate about web development, efficient, effective and accountable for leading successful, high quality projects.
  • Must Haves:
    • Proficiency writing current, clean, compliant, search-engine friendly HTML and CSS
    • Familiarity with JavaScript, and experience working with JavaScript libraries like jQuery
    • Proficiency with Photoshop, code editors or IDEs, and other web production tools
    • Strong written and verbal communication skills
    • Strong task management and prioritization skills
    • Live examples of work
  • Wish List:
    • Graphic design experience or an eye for front-end development
    • Experience working with Git version control system
    • Experience building responsive websites
    • Knowledge of SEO best practices
    • Proficiency with Less and Sass CSS preprocessors
    • Proficiency with front-end programming using JavaScript and jQuery
    • WordPress, PHP, .Net MVC, and Coldfusion experience preferred
