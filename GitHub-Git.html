GitHub and Git

On 6 Feb 2017

GitHub

I already had a GitHub account from my CIT15 last semester. I also followed Rio's instruction video so there was no problem setting up the page. I remember not really understanding how GitHub works back then, but after going through the HTML trainings and researching for the past few weeks, the web hosting feature that GitHub offers makes sense to me now. Furthermore, a friend explained to me that GitHub is a great tool for when you're working on a group project and each person is responsible for a certain part. They can upload their part on GitHub and take turn working on the project, and if someone happens to make a grave mistake, you can retrieve an older version of the project file for recovery purpose.

Git

I've sucessfully downloaded and installed Git Bash on my system. I've seen a lot of tutorial video where the lecturer uses Git for version control, but I still don't really get how Git works (I can look up the Cheat Sheet for commands guidance though). I guess I learn more easily through practical exercise rather than just checking things out without a real goal. What I've figured out so far is the command git to display a list of common Git commands; clear is used to clear previous lines and pull up a new blank page; cd is used to navigate between folders and directories (for example cd C:\User will bring you to the User folder, cd .. will bring you back to the upper directory). I hope to get more used to Git once we actually apply it in a project.
