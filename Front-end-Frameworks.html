Other Front-end Frameworks

On 17 Feb 2017

MUELLER

by Aura Le

MUELLER is a modular grid system for responsive/adaptive and non–responsive layouts, based on Compass. Users will have full control over column width, gutter width, baseline grid and media queries. It offers full control, clean code, maximum flexibility. Like many other popular front end frameworks, MUELLER is also free to use.

MUELLER doesn't have a default 12-columns grid like Bootstrap. Instead, it let users decide for themselves how they want their page layout to be like, so I believe this framework isn't as user-friendly as Bootstrap and requires a certain level of familiarity with the grid system. On the bright side, it's more flexible in the sense that the designer isn't restricted to 12-columns system. They can make a design depending on the targeted device (e.g 24-columns for desktop, 6-columns for mobile device, etc.). Since MUELLER does requires coding, once the user understands it, they can define grid that adapts to every device with ease.

PURE

by Yvetter Avalos

For this assignment, I decided to go and research more into Pure.css which is a small framework made up of CSS modules. I was really fond of CSS last semester in CIT and really loved learning it. Using Pure, you can easily create menus, tables, and other features. Pure is easy to create really great looking responsive layouts for all screen sizes. Its minimal styling ensures that you can customize the look of elements to suit your needs without any effort. Pure is extremely lightweight and it is ensuring fast loading times even on mobile devices. Pure's framework makes it easy to design websites. You can use Pure in every web project and the good thing about it is, it is ridiculously small and clocks in at 3.8KB. This framework is ideal for web developers who are focused on creating responsive and fast mobile websites. The down fall to this framework is that it is CSS only, and does not support JQuery or JS plugin.
