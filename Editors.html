Visual Studio Code - Text editor

On 26 Jan 2017

I heard of Sublime Text and Atom from friends, and I tried to do some little research to see which one is better to use. I came across several comparison articles or threads, where people usually compare Sublime Text, Atom, VS Studio and Brackets. It appears to me that a lot of people prefer VS Studio over other editors because "better IntelliSense than any other". With that said, I decided to go with VS Studio. I haven't been able to try out all of its special features yet, but the "auto-complete" IntelliSense is indeed really nice, and they also offer either a full-install version or portable (unzip-and-run) version for you to download.

Visual Studio Code tutorial

VS Code works on multiple platform (OSX, Mac, Windows, Linux) and works with many languages (JavaScript, C#, etc.). It is fast, powerful, and cross-platform. It is built on Electron (Atom Shell)and will update automatically.
