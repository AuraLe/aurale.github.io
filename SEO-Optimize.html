Implementing optimization

On 12 Feb 2017

Domain name should contain keyword for the site (except when building a brand around an uncommon word) and should have at most 3 short words.

Avoid uncommon top-level domain. Prioritize getting a .com domain, as it is for commercial use. Make the domain memorable but not confusing. Avoid using hyphen in the domain name when possible, if not possible then only use one. Short domain names are always better than long.

Home page/landing page should grab visitors’ attention and contain important information about the site, like a magazine cover. Use single prominent heading. Group contents into small blocks of info for easier view. Match user’s tone to website design.

CAT (Call to Action): button or link that drives users deeper into the site.

Above the fold: content that fit into viewer screen w/o scrolling down.

Product page inform users about a product, and can be linked back to by outside parties. So you should implement breadcrumb to product pages for better navigation (and more info for search engines). Give users option to share product page on social media since it is a good way to get linked back and increase traffic. Create a template for product page, including: brand name in title, product category, &ersand; business name. Description for each product should be unique and well-written.

Websites without mobile-friendly version may not drop in ranking, but may have lower traffic. Use Google’s tool to check if your site is mobile-friendly or not. Text sizes and button sizes should be appropriate for finger nagivation too, not just mouse. Also avoid pop-up windows.
