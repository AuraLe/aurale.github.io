Incorporating SEO Tools

On 13 Feb 2017

A-B-Test

This is a split test that shows a slightly different version of website to a portion of usual traffic. Recommended tool: Optimizely. Goal: Making a tweak to website and compare the two versions to see which one performs better. You can save multiple different versions. Setting up Optimizely seems complex at first, but it’s a very useful tool to look deeper into traffic statistics so that you can customize your website for better user experience.

Google Analytics

Google Analytics let website owner knows all sorts of information regarding site traffic: number of visits, average page visits, language &ersand; location of users, devices that users use to visit site, etc. GA also let owner know which keywords are used that lead to the website (can’t track users who log into their Google account) >> GA is a powerful to analyze traffic demographic for better site design that suits the targeted audience.

Heatmapping tool

It tracks how far down the page a user is scrolling, and provides color code to indicate which part of the page is more attention-grabbing. Recommended tool: Crazy Egg (paid service with free trial). The tool’s snapshot provides info on website screenshot with indicators >> Less popular parts will need improvement.

Moz

Moz is a suite of SEO products designed for large-scale sites. Paid service with 30-days trial. Moz offers SEO tools that are similar to Google Analytics, Optimizely and Crazy Egg. Other tool: Page Grader, grading on how well a page is optimized for a certain keyword. Moz is more pricey than other tools, but worth it because it can help increase the business sale via in-depth analytics.
