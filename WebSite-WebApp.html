Web Sites vs Web Apps

On 16 Jan 2017

Site vs App

Web site

Mainly serves the purpose of providing information to users. There is hardly any interaction between users and the site, except for simple tasks like subscribing via email or search.

UCreative
This is a website about design and photography. It has articles on current trend, tips, news, rules of design/photography, and provides some resources for designing. If you're interested in graphic design, it's worth spending some time looking through this website.

Web app

Mainly includes interactive activities with visitors. It usually receives information from users and allows them to view personally customized contents. Web app requires more maintenance and security than a usual website.

Adobe Color CC
This is an app by Adobe that helps you with color palette. It allows you to create colors using color wheel and slider. It can also come up with a color theme based on an image that you upload. If you don't feel like you can make a good color combination, you can browse through premade themes that other people created.

Built it and they will come

I'd like to build a website for my guild in an MMORPG that I'm playing, because a forum post doesn't seem to be enough. We are quite big comparing to other guilds, and since we have been founded for almost two years, there are a lot of memories and history to the guild. I think it'd be great to include all our info, achievements, as well as a long list of members and their special traits, into a website.
