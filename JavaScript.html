JavaScript

On 9 Feb 2017

JavaScript is the programming language of the web. In a webpage, HTML describes the structural content; CSS is responsible for page layout, styling and formatting; while JavaScript controls how certain elements on the page behave.

JavaScript can do many things to the page content. It can change the HTML content upon certain condition is met (clicking on a button for example). It can also change the link leading to an image, thus replacing the old image with another one. It can even influence the styling of the page, meaning making modification to CSS.

In HTML, JavaScript code must be inserted between <script> and </script> tags. A JavaScript function is a block of JavaScript code, that can be executed when called" for. For example, a function can be called when an event occurs, like when the user clicks a button. Scripts can be placed in the <body>, or in the <head> section of an HTML page, or in both.

Scripts can also be placed in external files, which becomes more practical when the same code is used in many different web pages. To use an external script, put the name of the script file in the src (source) attribute of a <script> tag:

<!DOCTYPE html>
<html>
<body>

<script src="myScript.js"></script>

</body>
</html>
 

JavaScript can display data in a few different ways.

innerHTML
Writing into an HTML element using the document.getElementById(id) method. Changing the innerHTML property of an HTML element is a common way to display data in HTML.
document.write()
Writing into the HTML output, usually for testing purposes. Using document.write() after an HTML document is fully loaded, will delete all existing HTML.
window.alert()
Writing into an alert box.
console.log()
Writing into the browser console, mostly for debugging purposes.

JavaScript is a programming language, functioning on the "instructions" and "execution" basis. For a programming language, those sets of instructions are called "statements". JavaScript statements are separated by semicolons, and are composed of: Values, Operators, Expressions, Keywords, and Comments.
